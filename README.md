** Место ** 

*Учебный сайт с информацией о местах России*

Примененные технологии:  рефакторинг кода с применением ООП стиля программирования, созданы классы для хранения и обмена информацией, функции преобразованы в методы классов, выложен на gh-pages, общение с сервером, сборщик через webpack, валидация форм, JavaScript, адаптивная верстка, БЭМ, html, css*

[ссылка на проект](https://eugenyptichkin.github.io/mesto-project/index.html)
